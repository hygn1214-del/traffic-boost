# TrafficBoost Pro

로그인과 링크 입력이 있는 **트래픽 부스터 데모 웹앱**입니다.
정적 HTML 한 파일(`index.html`)로 동작하며, 별도의 빌드나 서버가 필요 없습니다.

> ⚠️ 이 프로젝트는 **화면 시연용 데모**입니다. 실제로 외부에 트래픽을 전송하지 않으며,
> 진행률·통계·로그는 모두 브라우저 안에서 생성되는 시뮬레이션입니다.

## 데모 계정
- 아이디: `admin`
- 비밀번호: `1234`

## 로컬에서 열기
`index.html`을 브라우저로 열면 됩니다.

## Vercel 배포
정적 사이트이므로 프레임워크 설정 없이 그대로 배포됩니다.

```bash
vercel        # 미리보기 배포
vercel --prod # 프로덕션 배포
```
